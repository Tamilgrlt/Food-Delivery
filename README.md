## 📌 Overview

A modern and responsive Food Ordering Web Application built using HTML, CSS, and JavaScript. Users can browse food items, explore categories, and place orders through an intuitive interface.

---

## 🚀 Features

- 🍕 Responsive Design
- 🔍 Food Search
- 🍔 Food Categories
- ⭐ Popular Dishes
- ❤️ Favorites
- 🛒 Shopping Cart
- 💰 Dynamic Price Calculation
- 📱 Mobile Friendly
- 🎨 Smooth Animations

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Font Awesome
- Google Fonts
